Estructura if-else-if.
//Casi siempre se declara una variable gloibal para trabajar con ella.
'let x'
if (condición) {
    ¿console.log()?
}
else if (condición) {
    ¿console.log()?
}
else{
    ¿console.log()?
}

-> Sentencia switch.
//Casi siempre se declara una variable gloibal para trabajar con ella.
'let x'
switch ('x') {
    case '' :
    ¿console.log("string");
    break;
    ...
    default:
    ¿console.log("string");
    break; 
    }
//Para indicar que la ejecución de las sentencias debe cortarse cuando se cumpla, se coloca 'break;'.
//Cuando terminemos de establecer los casos que necesitemos, debemos colocar el valor 'default:'. Nos permite establecer un valor por defecto, para los casos en que el valor asignado no empate con los de ninguno de los casos del bloque de código.

-> Operador terniario.
'let x' = (condicion) ? 'valor1|true' : 'valor2|false'
//El valor de la izquierda representa un valor falso y, el de la derecha, uno verdadero.
