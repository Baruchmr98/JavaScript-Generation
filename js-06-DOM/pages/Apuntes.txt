1.  'document.getElementById' Trae elementos por ID
2.  'document.getElementsByClassName' Trae elementos por Class
3.  'document.getElementsByTagName' Trae elementos por Etiquetas

4.  '.innerText' Extrae el texto que vive dentro del elemento. Además, puede ocuparse para modificar el texto de un elemento desde el DOM
5.  '.length' Devuelve la longitud de elementos de la clase 
6.  'typeof' Muestra el tipo de objeto que sea.
7.  '.style' Modifica el estilo de un elemento
8.  '.textContent'
